## 1. 目标市场

### 1.1 ICP（理想客户画像）

| 画像维度 | S级客户 | A级客户 | B级客户 |
|---------|---------|---------|---------|
| **行业** | 连锁零售 / 4S店集团 / 金融保险 | B2B SaaS / 房产中介 / 教育 | 各类中小企业销售团队 |
| **销售团队规模** | 500+ | 50-500 | 10-50 |
| **新人月流动率** | >10%（强痛点） | 5-10% | ~5% |
| **培训预算区间** | ¥20-50万/年 | ¥5-20万/年 | ¥1-5万/年 |
| **关键决策人** | 销售VP + CIO | 销售总监 + 培训经理 | 老板/销售主管 |
| **技术偏好** | 私有化部署 / SSO / 数据不出域 | SaaS + 企业微信集成 | 开箱即用、低门槛 |

### 1.2 TAM / SAM / SOM 估算

| 层级 | 定义 | 估算逻辑 |
|------|------|---------|
| **TAM** | 中国境内有内部销售培训需求的企业总数 | 中国连锁零售企业约5万家 + 保险代理人机构约2000家 + B2B SaaS企业约2万家 → 约10万家企业 |
| **SAM** | TAM中愿意采用SaaS培训方案的企业 | 行业报告显示企业培训SaaS渗透率约15%，SAM ≈ 1.5万家企业 |
| **SOM** | SAM中我们24个月内可触达并服务的 | Phase 2目标50家付费客户 + 渠道覆盖200家 → SOM ≈ 250家（占SAM 1.7%） |

## 2. 种子客户获取

### 2.1 第一批5家种子客户画像与获取渠道

| 序号 | 画像 | 获取渠道 | 转化路径 |
|------|------|---------|---------|
| 1 | 连锁零售企业（A级，200+门店销售） | HR Tech峰会 + 销售培训白皮书内容营销 | 峰会Demo → 14天免费试用 → 年度签约 |
| 2 | 保险公司电销团队（S级，300+电销坐席） | 保险科技行业沙龙 + 个人network | 行业沙龙演示语音陪练 → POC验证 → 私有化部署签约 |
| 3 | B2B SaaS企业（A级，60人销售团队） | 销售赋能社群 + 公众号内容分发 | 社群触达 → 免费试用 → 标准版/专业版签约 |
| 4 | 房产中介品牌（A级，100+经纪人） | 企业培训SaaS代理商渠道 | 代理商推荐 → POC → 专业版签约 |
| 5 | 4S店集团（S级，10+门店销售） | 行业峰会赞助 + 定向BD | 高层对接 → 定制场景Demo → 企业版签约 |

### 2.2 试点策略（POC流程）

```
第1周：部署开通（SSO配置 + 组织架构导入 + 2个场景模板配置）
第2周：培训经理+3名学员试用文字陪练（L1-L2），培训经理验收评估报告
第3周：扩展到1个完整团队（10-15人），下发培训任务，管理者查看团队看板
第4周：POC总结汇报（陪练数据 vs 基线指标 + ROI预估），推动采购决策
```

POC关键成功指标：①POC期间团队完成率≥80%；②培训经理主观满意度≥4/5；③管理者认可"可替代或减少部分线下培训"。

### 2.3 种子客户激励方案

- **首批5家种子客户享首年5折**（标准版¥7,500、专业版¥29,000、企业版¥99,000）
- 种子客户专属场景定制1个免费（价值¥30,000）
- 联合案例白皮书：将种子客户打造为行业标杆案例，品牌联合曝光
- 产品共创权益：优先体验新功能，直接对接产品团队反馈

## 3. 定价与包装

### 3.1 免费试用到标准版的升级路径设计

| 阶段 | 内容 | 限制 | 转化触发器 |
|------|------|------|-----------|
| 免费试用（14天） | 全功能开放 | ≤10个学员席位、3个基础场景 | 试用到期 / 学员席位耗尽 / 需要专属场景 |
| 标准版（¥15,000/年） | 30席位 + 全部场景 + 自定义模板 + SSO | 无语音 / 无高级看板 / 无CRM集成 | 需要语音模式 → 升级专业版；需要私有化 → 升级企业版 |
| 专业版（¥58,000/年） | 100席位 + 语音 + 高级看板 + 任务下发 + CRM | 无私有化部署 | 团队规模超100人或需要私有化 → 升级企业版 |
| 企业版（¥198,000/年） | 500席位 + 私有化 + SLA 99.9% + 客户成功经理 | — | 按需定制 |

**升级触发节点**：①学员席位达上限（自动弹窗引导升级）；②点击语音模式入口（引导开通语音增值包或升级专业版）；③数据看板提示"高级分析功能需专业版"。

### 3.2 语音增值包的销售时机

- **何时推**：客户完成文字陪练核心闭环（≥10次团队陪练）后，且团队中有电销/电话沟通岗位。
- **怎么推**：在陪练场景选择页展示"语音模式（推荐）"标签 → 点击弹出10分钟免费语音体验 → 体验后引导购买。客户成功经理在月度复盘会上主动推荐。
- **捆绑策略**：语音增值包可独立购买（¥20,000/年）或随专业版打包（专业版已含语音，无需额外购买），鼓励直接升级专业版。

### 3.3 大客户定制化报价策略

- S级客户报价模板：企业版基础（¥198,000）× N（按团队规模线性扩展）+ 私有化部署（¥150,000起）+ 专属场景定制（¥30,000/个 × N个）
- 定价弹性空间：首年折扣（8-9折）、多年合同折扣（2年9折/3年8折）、额外场景定制赠送
- 报价锚定：以得助智能金融行业定制（¥30-80万/年）为高价锚、Megaview（¥10-50万/年）为中价锚，企业版¥198,000处于中高价区

## 4. 渠道策略

### 4.1 直销（B级客户）

- 销售团队自己卖，1个AE + 1个SDR，月均触达50家企业
- 核心销售动作：内容营销获客 → SDR筛选 → AE Demo演示 → 14天免费试用 → 关单
- B级客户决策链短（老板/主管直接拍板），直销效率高

### 4.2 渠道（A/S级客户）

- 企业培训/HR SaaS代理商：已服务企业培训客户的代理商具有天然的信任关系和客户池，可快速触达A/S级客户
- 分成模式：首年30%渠道佣金 + 续费15%佣金
- 代理商赋能：提供标准化Demo环境、产品培训、联合营销物料

### 4.3 合作伙伴（平台生态）

- **企业微信/飞书/钉钉应用市场**：上架标准版SaaS应用，借助平台流量触达已使用IM的企业客户
- **CRM厂商合作**：与销售易、纷享销客等CRM厂商做集成合作（PRD Phase 2），嵌入CRM中的"销售培训"模块，双向引流
- **HR SaaS集成**：与北森、Moka等HR系统对接，在入职流程中嵌入AI陪练，从HR场景切入

## 5. 市场推广

### 5.1 内容营销

- **销售培训白皮书**（季度发布）：《AI陪练如何缩短新人上岗周期》《连锁零售培训标准化实践》《销售团队能力数据化管理指南》
- **行业案例**（种子客户联合发布）：客户名称 + 使用前后数据对比 + 客户证言
- **分发渠道**：公众号、知乎、36氪、人人都是产品经理、销售管理类社群

### 5.2 社区运营

- **销售管理者社群**：微信群/飞书群，每周分享销售培训干货，定期邀请种子客户分享使用心得
- **社群转化**：免费试用专属链接 → 社群内答疑 → 满意度调查 → 推荐升级
- 目标：6个月内社群规模达500人（销售培训经理/销售VP），月活跃率≥30%

### 5.3 行业峰会

- **参会**：HR Tech峰会、销售赋能峰会、连锁零售数字化峰会、保险科技大会
- **形式**：展位Demo演示 + 主题演讲（需要种子客户案例） + 会后BD跟进
- **预算**：年度2-3场重点峰会，单场预算¥30,000-50,000（含展位+差旅+物料）

## 6. 里程碑与KPI

### 6.1 关键里程碑（规划）

| 时间 | 里程碑 | 核心交付 |
|------|--------|---------|
| M1 | MVP上线 | 文字陪练闭环 + 5家种子客户POC启动 |
| M2 | Phase 1 + 首批付费 | 团队看板 + 培训任务下发 + 付费客户≥20家 |
| M3 | Phase 1.5 语音上线 | 语音陪练 + 付费客户≥30家 + 语音使用率≥20% |
| M4 | Phase 2 规模化 | CRM集成 + 私有化部署 + 付费客户≥50家 + NRR≥100% |
| M5 | Phase 3 扩展 | 视频陪练 + S级客户≥5家 |

### 6.2 核心指标

| 指标 | 6个月目标 | 12个月目标 | 24个月目标 |
|------|---------|----------|----------|
| 付费客户数 | ≥5（种子） | ≥20 | ≥50 |
| ARR | ¥200,000 | ¥800,000 | ¥5,000,000 |
| NRR（净收入留存率） | — | ≥90% | ≥100% |
| LTV/CAC | — | ≥3:1 | ≥5:1 |
| POC转化率 | ≥50% | ≥30% | ≥35% |
| 月活跃企业数 | ≥5 | ≥18 | ≥45 |
| NPS | — | ≥50 | ≥60 |

### 6.3 节奏说明

- 0-6个月：不追求规模，聚焦种子客户PMF验证和产品迭代
- 6-12个月：在验证PMF后启动渠道铺设和内容营销，从直销+渠道双线获客
- 12-24个月：规模化增长，渠道贡献≥50%新客，S级客户≥5家确立行业标杆地位
